Réseau — diagnostic avancé

Couche physique, adressage, stabilité ping, TDR câble, conflits ARP, VLAN tagging, duplex, MTU, débit, DNS, découverte LAN.

Outils et commandes utiles
  • Windows : ipconfig /all · ping -t IP · arp -a · tracert IP · pathping IP · netsh trace start capture=yes
  • Linux/macOS : ip a; ip r · ping IP · ip neigh · traceroute IP · tcpdump -i eth0 host IP
  • Découverte : nmap -sn 192.168.x.0/24 · scan ARP · LLDP côté switch · Advanced IP Scanner
  • MTU : ping -f -l 1400 IP (Win) · ping -M do -s 1400 IP (Linux)
  • Duplex : show interface status (Cisco) · ethtool eth0 (Linux)
  • Débit : iperf3 -c IP (si iperf dispo sur les deux bouts)
  • ONVIF : ONVIF Device Manager (Windows) · onvif-probe

Matériel — tests & logs

PoE, logs caméra, firmware, NTP, ONVIF, multi-stream, température capteur, capture réseau.

Outils et actions
  • Logs : export syslog/diag via WebUI ou outil fabricant ; relever erreurs réseau/RTSP/NTP
  • PoE : show power inline · TDR : test cable-diagnostics tdr
  • Flux : VLC rtsp://IP:554/stream · charge CPU/temp si visible · ping continu ping -t
  • ONVIF : ONVIF Device Manager pour vérifier profils S/T/G
  • Isolation : remplacement croisé — injecteur / câble court / port différent
